# 贡献者行为准则

## 我们的承诺

为了营造一个开放和友好的社区，我们作为贡献者和维护者承诺：无论年龄、体型、身体健全与否、种族、性别认同与表达、经验水平、国籍、个人外貌、宗教信仰或性取向如何，参与本项目和社区都将免受骚扰。

## 我们的准则

有助于营造积极氛围的行为包括：

- 采用友好和包容的语言
- 尊重不同的观点和经历
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表现出同理心

不可接受的行为包括：

- 使用与性有关的语言或图像，以及不受欢迎的性关注
- 蓄意挑衅、侮辱/贬低的评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确允许便发布他人的私人信息，如地址或电子邮箱
- 在专业场合被认为不适当的其他行为

## 我们的责任

项目维护者有责任澄清可接受行为的标准，并应采取适当和公平的纠正措施来回应任何不可接受行为。

项目维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、问题和其他贡献，并可暂时或永久禁止任何贡献者参与他们认为不适当、有威胁、冒犯或有害的行为。

## 适用范围

本行为守则适用于项目空间内以及个人代表项目或其社区的公共空间。当个人代表项目或社区时，应使用官方项目电子邮件地址、通过官方社交媒体帐户发布信息，或在线上或线下活动中担任指定代表。

## 报告与处理

如发现任何不当行为或骚扰，请通过 GitHub Issue 或直接联系维护者渠道进行报告。所有投诉将被审查和调查，并产生被认为必要且适合情况的回应。项目团队有义务对事件报告者保密。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) 2.0 版。
